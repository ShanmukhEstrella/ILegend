Legend-Kernel files are present here.

## Tool Requirements:
- Python >= 3.0.0
- Jupyter lab >= 4.0.0

## Package Requirements:
- pandas
- os
- re
- ast
- requests
- datetime
- json
- html
- ipyaggrid


## Usage:
Download the contents of this folder.
Then perform:

```bash
pip install .
python3 -m legend_kernel.install
```
The kernel will be installed in your jupyter environment!!
